# PHC AI Assistant - Aarogya

Offline AI Voice Healthcare Assistant for Indian PHCs.
See full documentation in implementation_plan.md.

## Quick Start
1. flutter pub get
2. Add llama.cpp submodule: git submodule add https://github.com/ggerganov/llama.cpp android/llama.cpp
3. Set model URL in lib/core/constants/model_constants.dart
4. flutter run --release
